# Pong

Riff on [this tutorial](https://www.geeksforgeeks.org/create-a-pong-game-in-python-pygame/).

## Minimum Requirements

- Python 3.11
- Pygame 2.5

## Installation

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

## Running

```bash
python pong.py
```
